# Clone direto (sem terminal, só cliques)

## Opção 1: GitHub + Vercel (RECOMENDADO)

### Step 1: Crie um fork do projeto
1. **No celular, clique aqui:**
   https://github.com/new

2. **Preencha:**
   - Repository name: `recuperai`
   - Descrição: "Agente financeiro IA para recuperação de receita"
   - Clique "Create repository"

3. **Você vai receber um URL tipo:**
   `https://github.com/seu-usuario/recuperai`

### Step 2: Upload dos arquivos (a forma mais fácil)
1. **Vá pro seu repo recém-criado**
2. **Clique o botão verde "Code" > "Upload files"**
3. **Descompacte o recuperai.zip no seu celular** (ou peça pra alguém descompactar no PC)
4. **Selecione todos os arquivos** (menos `node_modules`)
5. **Clique "Commit changes"**

### Step 3: Deploy automático
1. **Vá em https://vercel.com**
2. **Clique "Add New" > "Project"**
3. **Selecione o repo `recuperai`**
4. **Clique "Deploy"**

**Pronto! Em 2-3 minutos você tem:**
```
https://recuperai-seu-username.vercel.app
```

---

## Opção 2: Vercel + GitHub direto (ainda mais rápido)

**Se você quer CLONAR meu repositório público direto:**

1. **Clique aqui no celular:**
   https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Frecuperai-public%2Frecuperai&project-name=recuperai-meu&repository-name=recuperai

2. **Clique "Create"**

3. **Clique "Deploy"**

4. **Pronto** — seu app estará rodando em minutos

---

## Opção 3: Railway.app (com banco grátis)

**Se quer que os dados PERSISTAM (não some após 24h):**

1. **Vá em https://railway.app**
2. **Clique "Start a New Project"**
3. **Clique "Deploy from GitHub repo"**
4. **Conecta sua conta GitHub**
5. **Cria um repo `recuperai` como no Step 1 acima**
6. **Seleciona em Railway**
7. **Adiciona Postgres** (Railway oferece grátis)
8. **Deploy automático**

---

## Qual escolher?

| Opção | Velocidade | Persistência | Recomendação |
|---|---|---|---|
